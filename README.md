
# 🎄 3D 圣诞记忆树 (Merry Christmas Tree)

这是一个基于 Three.js 开发的互动 3D 圣诞树网页。它不仅是一个视觉节日装饰，更是一个可以承载你与朋友珍贵回忆的照片墙。你可以将它通过 GitHub Pages 发布，作为一个特别的圣诞礼物链接发送给他人。

## 🚀 立即访问

**[👉 点击这里查看我的圣诞树网页](https://doris-ss.github.io/merry-christmas/)**

---

## 📖 交互功能说明

- **手势控制**: 使用摄像头控制圣诞树！

  - **✊ 握拳 (Fist)**: 合拢圣诞树。

  - **🖐️ 张开手掌 (Open Palm)**: 将粒子散开成星空。

  - **👌 捏合 (Pinch)**: 选中并放大一张“照片”回忆。

  - **👋 旋转手掌 (Hand Rotate)**: 在散开模式下旋转视角。



* **✨ 沉浸模式 (Immersive Mode)**：
* 点击页面上的按钮可以进入/退出沉浸模式。该模式下会隐藏 UI 界面，让你更纯粹地欣赏圣诞树与照片。


* **🎵 音乐播放**：
* 点击右上角的 **Music: OFF** 切换为 **ON** 即可播放背景音乐。
* *温馨提示：受浏览器安全策略限制，音乐通常需要用户进入页面后手动点击才能开启。*

## 🖼️ 如何固定展示你的照片 (1-n 命名法)

如果你希望网页在打开时就自动显示你的照片，而不是空空如也，请按以下步骤操作：

1. **准备素材**：将你的照片放入项目根目录下的 `images/` 文件夹中。
2. **规范命名**：将照片重命名为连续的数字，例如：`1.jpg`, `2.jpg`, `3.jpg` ... `n.jpg`。
3. **修改代码**：
* 打开 `index.html`。
* 使用 `Ctrl + F` 搜索关键词：**“照片数量”**。
* 找到如下代码段，并将 `count: 10` 修改为你实际的照片总数：


```javascript
const PHOTO_CONFIG = {
    count: 10,       // 修改此处的数字为您存放的照片总数
    path: 'images/', 
    extension: '.jpg' 
};

```

## 💻 本地运行指南

如果你下载了 ZIP 压缩包并在本地电脑运行，请注意：

### 方案 A：直接打开 (基础模式)

1. 解压压缩包，双击 `index.html`。
2. **限制**：由于浏览器安全隐私策略，本地 `file:///` 协议往往无法自动读取文件夹内的 `images/`。
3. **临时使用**：你可以通过页面上的 **Add Memories** 按钮手动从电脑选择图片上传到树上（注意：刷新页面后这些图片会消失）。

### 方案 B：使用 VS Code 运行 (完美模式)

1. 使用 **VS Code** 打开本项目文件夹。
2. 在插件市场搜索并安装 **Live Server**。
3. 右键点击 `index.html`，选择 **"Open with Live Server"**。
4. 此时浏览器会以本地服务器模式运行，能完美自动加载 `images/` 里的所有固定照片。

## 🌐 发布到网站 (GitHub Pages)

将此圣诞树变成一个朋友随时随地都能打开的链接：

1. **上传文件**：将所有文件（含 `index.html`、`music` 文件夹、`images` 文件夹）上传到你的 GitHub 仓库。
2. **确保结构**：`index.html` 必须位于仓库的**根目录**（最外层）。
3. **开启服务**：进入仓库的 **Settings** -> **Pages**。
4. **配置分支**：在 **Build and deployment** 下，选择 `main` 分支并点击 **Save**。
5. **等待发布**：约 1 分钟后，顶部会生成一个 `https://yourname.github.io/...` 的链接，转发给朋友即可！
